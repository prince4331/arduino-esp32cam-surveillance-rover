# ESP32-CAM Surveillance Rover

Wi-Fi streaming rover exposing camera feed and motor controls through the ESP32-CAM web server.

## Original Folder
02_Robotics_and_Vehicles\ESP32CAM_Surveillance_Rover

## Hardware
ESP32-CAM AI-Thinker, L298N motor driver, four DC motors, 5V regulator, camera mount.
